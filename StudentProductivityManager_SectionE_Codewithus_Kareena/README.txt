Student-Productivity-Manager

Describtion:
This is a Java-based console application designed to help students manage their study goals, track progress, monitor exam preparation, and stay motivated.

Purpose:
Students often struggle with:
- Lack of consistency
- Comparing themselves with others
- Low motivation
- Poor progress tracking


This project was created to solve these problems by providing:
✅ Goal management  
✅ Progress tracking  
✅ Motivation support  
✅ Mental wellness awareness  
✅ Productivity monitoring 

The system allows users to:
- Add study goals
- Add exam goals
- Track study progress
- Generate productivity reports
- Detect negative/comparison thoughts
- Receive motivational messages
- Save and load data from files

The project demonstrates important Object-Oriented Programming (OOP) concepts in Java such as:
- Inheritance
- Polymorphism
- Abstraction
- Encapsulation
- Interfaces
- Exception Handling
- File Handling

How To Run:
To run the project, open the terminal in the project folder and compile the program using javac ui/Main.java, then start it with java ui.Main. After running, the console menu will appear with options like adding study goals, tracking progress, viewing reports, motivation, and exam goals
